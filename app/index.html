                📖 使用说明

                • 📋 待办:顶部输入框回车快速添加,双击空白详细创建;拖拽待办到左侧清单名可移动归属
                • 📅 日历:日/周/月视图切换;点日期格放大查看当天全部事项,双击快速新建;支持全天、起止时间、跨天与每天/每周/每月/每年重复
                • 🔲 四象限:按重要/紧急管理,拖拽调整优先级
                • ⏰ 提醒:设置日期+时间并勾选提醒,到点弹出系统通知(重复事件每次都会提醒)
                • 🤖 AI助手:对话可直接创建/查询/完成/删除待办、检查时段冲突,自动按内容归类清单;AI写待办自动排期;AI写周报汇总一周完成情况(需在 ⚙ 设置中配置 DeepSeek API Key)
                • ⏳ 倒计时·纪念日:支持公历/农历日期(农历自动转公历)、每年重复;点击条目可编辑
                • 🎨 窗口:⚙ 设置中可调壁纸、背景透明度、靠边自动隐藏(拖到屏幕边缘,鼠标顶到边缘弹出);📌 置顶;关闭进托盘,托盘右键可重置窗口位置
